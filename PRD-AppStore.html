一、模块概述

仿应用商店模块用于配置「系统设置 → 域名管理」中防封谷歌页域名 / 防封苹果页域名 / 仿谷歌页等下载落地页所引用的页面样式模板。页面高度仿真官方应用商店的视觉与交互，让用户误以为进入 Google Play / App Store，从而提升信任感、提高 APP 下载转化。

页面属于下载页类落地，唯一核心功能为「下载 / 安装 APP」；其安装交互与《安装引导模块 PRD》的下载组件完全共用。

项目说明
菜单位置商户后台 → 推广渠道管理 → 仿应用商店
模块产出下载落地页的样式模板，供域名管理中防封谷歌页 / 防封苹果页 / 仿谷歌页等下载落地域名引用
页面类型仿谷歌商店页、仿苹果商店页（按模板类型区分样式）
落地关系域名管理中相应下载落地域名引用本模块模板；用户经推广渠道链接，或经安装引导 iOS-Safari 流程进入
配置粒度以商户为单位，按模板逐条配置

3.4 评分星级与分布条算法

星级：5 颗星按 APP评分 ÷ 5.0 的比例点亮（支持半星）。如评分 4.0 亮 4 颗，评分 3.1～3.9 亮 3 颗半。

评分分布条：因「平均分」无法唯一反推 5 档分布，前端按固定算法根据 APP评分动态生成，使加权平均恰等于配置评分（改评分时分布条自动匹配，无需后台额外配置）：

设配置评分为 r。固定「低分块」内部比例 4★:3★:2★:1★ = 10:4:3:3（即 0.5 / 0.2 / 0.15 / 0.15，内部均值 m≈3.05）。
5★ 占比：p5 = (r − m) / (5 − m)；其余 (1 − p5) 按上述比例分配给 4★～1★；r < m 时 p5 取 0。
条形长度：各档占比相对「最大档」归一，5★ 满宽。

示例（r=4.8）：5★≈89.7% / 4★≈5.1% / 3★≈2.1% / 2★≈1.5% / 1★≈1.5%，加权平均正好 4.8。

四、下载与安装交互

4.1 交互总则

用户通过带参推广链接 / 域名管理配置的落地域名进入本模块的下载页（仿谷歌页 / 仿苹果页）后，页面仅提供下载安装能力。点击 CTA 或页面内各下载触发点后，系统依据【用户设备类型】与【后台安装 / 下载配置（安装类型、安装包等）】触发对应安装流程：安卓设备按安卓安装类型（PWA / APK / PWA+APK）流转，iOS 设备按苹果安装方式（保存桌面）流转，并结合浏览器环境（Safari / 非 Safari、是否支持 PWA）进入相应分支。

统一触发点：CTA 按钮，以及数据安全「查看详情」、评论「有用/没用」、「查看全部评价」、相似游戏标题箭头、底部「在 App Store 下载」按钮，点击后均触发同一套下载流程。

设备进入页面时点击 CTA / 触发点后
安卓正常展示页面下载确认弹框（加载状态 → 就绪状态，见 4.2）→ 按安卓安装类型走后续（见 4.3）
iOS加载时即判定浏览器：若非 Safari，页面加载同步弹出复制链接弹框下载确认弹框（加载状态 → 就绪状态，见 4.2）→ 「添加到主屏幕」说明浮层（见 4.4）

4.2 下载确认弹框的三种状态

下载确认弹框居中展示，固定含标题「快速安装」、速度标签、「已激活」状态标签；其进度区在交互过程中呈现三种状态：

状态进度区表现触发与流转
① 就绪状态展示绿色「安装」按钮加载状态完成后进入；等待用户点击「安装」
② 加载状态圆弧进度动画（0%→100%）点击 CTA / 触发点后进入，表示极速安装准备；达 100% 后切换为就绪状态
③ 下载状态
（仅安卓 APK）		「安装」按钮区替换为线形进度条（0%→100%）就绪状态下点击「安装」并进入 APK 下载时进入；满格后恢复为绿色「安装」按钮，并自动完成 APK 下载

状态流转：点击 CTA → ② 加载状态 → ① 就绪状态 →（安卓 APK）③ 下载状态 → ① 就绪状态。下载状态仅出现在两个商店页的安卓设备 APK 下载场景；全过程弹框常驻不关闭（见 4.5），APK 下载成功不额外弹出提示浮层。

4.5 弹框层级与常驻规则

常驻：下载确认弹框弹出后始终不关闭、保持显示，后续弹框叠加于其上层。

层级（由低到高）：商店页 < 下载确认弹框 < 「添加到主屏幕」浮层 < PWA 系统提示 / APK 确认弹框 / 复制链接弹框 < 轻提示(toast)。

PWA 系统提示弹出时带独立遮罩，遮罩盖住原页面及下载确认弹框，PWA 提示本身位于最上层；即 PWA 提示弹出时其下所有信息都处于遮罩之下。APK 确认弹框、复制链接弹框自带暗色背景，同样覆盖于下载确认弹框之上。

4.6 与安装引导模块的复用关系

本模块与《安装引导模块》共用同一套下载组件：下载确认弹框、PWA 系统提示（含遮罩）、APK 确认弹框、复制链接弹框（Chrome / Safari）+「复制成功」提示、「添加到主屏幕」说明浮层。

《安装引导模块》中「下载按钮 / 底部浮层」在 iOS·Safari 点击安装后跳转的页面，复用本模块的仿苹果页；页内点击安装后弹出的下载确认弹框与「添加到主屏幕」说明浮层，即本模块的同一套组件。
